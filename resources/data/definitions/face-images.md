# 人脸图像数据集

## 1. 数据集概述
本数据集包含大量人脸图像，可用于人脸识别、表情分析、属性分类等计算机视觉任务。图像采集自多种环境，涵盖不同光照、角度和背景，并标注了性别、年龄范围、表情等属性。

## 2. 目录结构
```
face-images/
├── images/
│   ├── 00001.jpg
│   ├── 00002.jpg
│   └── ...
├── metadata.csv
└── README.md
```

## 3. 数据格式

| 文件 | 格式 | 大小 | 说明 |
|------|------|------|------|
| images/*.jpg | JPEG | 约 10,000 张 | 原始人脸图像，分辨率 256x256 |
| metadata.csv | CSV | 500 KB | 与图像对应的属性标注文件 |

## 4. 数据 Schema

| 字段 | 类型 | 说明 |
|------|------|------|
| image_id | string | 图像文件名（不含扩展名） |
| gender | string | 性别标注（male/female） |
| age_group | string | 年龄段（child/youth/adult/senior） |
| expression | string | 表情分类（neutral/happy/sad/angry/surprised） |
| has_glasses | bool | 是否戴眼镜 |
| lighting | string | 光照条件（indoor/outdoor/dim） |

## 5. 数据来源
图像采集自公开人脸数据集（如 CelebA、LFW、自建数据集），并经过人工清洗与标注。所有图像的使用遵循原始许可协议，仅限研究与开发用途。

## 6. 使用场景
- 人脸检测与识别模型训练/验证
- 表情识别与情绪分析
- 人脸属性编辑与生成
- 光照、姿态不变性研究
- 隐私保护下的人脸数据增强

## 7. 质量评估
- 标签准确率：经交叉验证，属性标注准确率 > 95%
- 图像质量：人工剔除模糊、遮挡严重、非正面人脸图像
- 数据平衡：各属性类别基本均衡，无严重倾斜
- 去重处理：使用感知哈希去除近重复图像

## 8. 访问权限
本数据集为内部使用，访问需项目负责人审批。如需公开，应遵循数据脱敏与授权流程。
